# name

extends 100-character message limit of 1.8.9 up to 256 characters; queues regular messages & commands; adds long message splitting; increases default message history length; adds support for command & regular prefixes


[IMAGE]


## installation

you can either install the `.jar` from the releases or build the mod yourself by cloning the repository and running:

### windows

```bat
.\gradlew.bat build
```

### linux

```bash
./gradlew build
```

the built `.jar` will be inside `./build/libs` directory in both cases
